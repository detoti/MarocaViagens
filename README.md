# MarocaViagens - 2.0

Depois de estudado autolayout e constraints no app Maroca Viagens, decidi que era hora de implementar novas features e funções no app.

- Novas telas foram criadas, como: Cadastro e Pagamento
- Animações incluídas
- Validações de Text Fields
- CocoaPods para validação de CPF
- RegEx para validação de E-mail
- ViaCep para validação de Endereço completo (em desenvolvimento)
